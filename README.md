# Advanced-Ultrasonic-Proximity-Detection-and-Reverse-Parking-Assistance-System
Designed and implemented an Advanced Reverse Parking System using ARM7 (LPC2129) and CAN protocol for reliable vehicle node communication. Integrated ultrasonic sensor (HC-SR04), LCD display, and buzzer for real-time obstacle detection and driver alerting. Programmed in Embedded C with interrupt handling, GPIO control,.
